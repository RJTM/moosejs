Uploading a contest zip file

Zip file structure is described below

Contest.zip
  • contest.json Main configuration file for the contest
  • tasks/
    • Task 1 One folder per task. Folder name should be the task code
      • Task 1.pdf Task statement in pdf format.
      • Subtask 1 One folder per subtask. Folder name should be Subtask number
        • One pair of files per testcase
        • 1.in
        • 1.out
        • ...
        • n.in
        • n.out
    • ...
    • Task N
